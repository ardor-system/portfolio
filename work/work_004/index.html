ノーツエディタ

※音量にご注意下さい。

ジャンル データ編集ツール
プラットフォーム Windows
開発 Visual Studio (MFC)
使用ライブラリ OpenGL(譜面の描画) / Boost(JSONの読み書き) / Ogg Vorbis(音声ファイルの再生)


概要

リズムゲーム等で使用するための譜面データを作成・編集するためのツールです。



操作

編集 キーボード・マウス操作


機能

  • 編集対象の楽曲フォルダの選択。
  • 各種データの確認・編集。
  • マウスのクリック操作でノーツの配置等が編集可能。
  • 独自のフォーマットのデータで出力可能。
  • BPMの解析が可能。
